# 🚀 高级正则 PDF 阅读器 (Advanced Regex PDF Reader)

一款基于 Python (PyQt5 + PyMuPDF) 开发的桌面级 PDF 阅读与编辑工具。专为需要深度解析长文本、处理复杂匹配规则以及重组 PDF 结构的高阶用户打造。它不仅拥有原生级的高清渲染，更在底层实现了严苛的性能优化，即使面对数千页的超大 PDF 也能保持极致丝滑。

## ✨ 核心特性

### 🔍 极客级正则搜索
* **全文档跨行匹配**：利用正则表达式突破物理换行限制，精准捕获跨行文本。
* **智能图层高亮**：独创的底层渲染引擎，全局匹配项铺设“护眼黄”底色，当前激活项自动覆盖“聚焦橙”顶层高亮，视觉层次分明且永不遮挡。
* **搜索预设管理**：将常用的复杂正则保存为预设字典。具备自动记忆功能，重启软件也能恢复上一次的工作流。

### ⚡️ 极致的性能优化 (丝滑不卡顿)
* **动态懒加载 (Lazy Loading)**：在连续滚动模式下，自动为非可视区域生成极低内存占用的“白色骨架”。只有当页面滑动到屏幕范围内时，才瞬间进行高清图像与字符坐标的渲染。
* **防抖引擎 (Debounce)**：对频繁的缩放操作引入了毫秒级的冷却机制，彻底杜绝 UI 线程阻塞与假死。
* **按需字符解析**：废弃了传统的全量预加载，仅在鼠标点击页面的瞬间才提取底层的矢量字符坐标，让大文件的开启速度产生质的飞跃。

### 🖱️ 生产力交互
* **精确文字框选**：通过底层物理坐标映射，支持像原生浏览器一样极其精准的字符级拖拽高亮与右键/快捷键复制。
* **双模式无缝切换**：支持“单页聚焦”与“连续滚动”双模式，切换瞬间完成。
* **原生书签编辑**：自动读取 PDF 内部 TOC（大纲）。支持添加、删除，以及**双击直接修改书签名称**，并双向同步至底层 PDF 文件。

### 📑 内存级页面重组
* **非破坏性重排**：支持输入如 `1, 3-5` 等复杂指令，瞬间将所选页面剪切并插入到指定位置。
* **页面删除**：一键剔除多余页面。
* *(注：以上操作均通过 `doc.select()` 在内存中瞬间重组，速度极快，且保存前绝不污染原文件。)*

### 🛡️ 数据安全与生命周期
* **修改状态追踪**：具备专业生产力软件的防呆机制。无论是增删书签、重排页面还是删除页面，只要发生改动，在关闭文件或退出程序时均会触发安全拦截，提醒保存。

## 🛠️ 安装与运行

### 环境要求
* Python 3.8+
* 操作系统：Windows / macOS / Linux

### 依赖安装
请在终端中运行以下命令安装所需的核心依赖：
```bash
pip install PyQt5 PyMuPDF
```
### 启动程序
```Bash
python pdfsearch.py
```
### 📖 快捷键与使用技巧
* Ctrl + O : 快速打开本地 PDF 文件

* Ctrl + S : 增量保存（极速覆盖原文件）

* Ctrl + Shift + S : 另存为新文件

* Ctrl + W : 关闭当前文件

* Ctrl + F : 快速聚焦到右侧正则搜索框并全选已有内容

* 连续页面编辑：在“文档”菜单中使用“重新排列页面”或“删除页面”时，支持使用连字符，例如：输入 2, 5-8, 12 即可精准定位多个非连续区间。

* 保存预设：下拉菜单的预设支持修改覆盖。如果想基于现有预设修改，选中它，在下方文本框修改后，再次点击“新增预设”并输入同名即可。

### 🏗️ 技术栈说明
* GUI 框架：PyQt5 - 提供桌面级原生流畅的交互界面。

* PDF 解析引擎：PyMuPDF (fitz) - 提供底层高性能的矢量图像渲染、字符边界框 (BBox) 提取与内存级 PDF 结构重组。

* 本地存储：利用 QSettings 写入 .ini 格式的轻量级便携配置文件，实现绿色无污染的设置记忆。
