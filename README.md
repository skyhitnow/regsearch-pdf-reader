# 高级正则 PDF 阅读器

这是一款基于 PyQt5 和 PyMuPDF 开发的本地 PDF 阅读与轻量编辑工具，专为处理长文本和复杂的检索需求而设计。

## 核心功能清单

### 1. 基础阅读与操作
* **双模式显示**：支持“单页显示”与“连续滚动”两种阅读模式的无缝切换。
* **画面缩放**：支持通过按钮或滑块自由调整 PDF 的放大与缩小比例。
* **精准复制**：支持在 PDF 页面上使用鼠标拖拽，进行极其精准的字符级文本框选与复制。
* **快捷导航**：支持上一页/下一页快速切换，以及输入特定页码进行瞬间跳转。

### 2. 高级正则搜索
* **跨行匹配**：支持使用正则表达式对全文档进行搜索，完美解决传统阅读器无法跨行检索的问题。
* **高亮定位**：所有搜索结果在页面上以黄色底色标记，当前选中的结果会自动叠加显眼的深橙色，并在右侧列表中同步展示。
* **正则预设管理**：支持将常用、复杂的正则表达式保存为预设模板；支持修改与删除，并具备自动记忆功能，下次打开可直接调用。

### 3. 书签 (大纲) 管理
* **自动读取**：加载文件时，自动提取并显示 PDF 原生目录树。
* **快捷编辑**：支持一键添加当前页面为书签、删除指定书签。
* **直接重命名**：在书签列表中，双击任意书签名称即可直接进行修改，并双向同步至底层文档。

### 4. 页面编辑与重组
* **重新排列页面**：支持输入区间指令（如 `1, 3-5`），将指定的多个页面剪切并插入到文档的任意位置。
* **删除指定页面**：支持批量清理文档中不需要的页面。
