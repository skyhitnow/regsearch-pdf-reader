# 高级正则 PDF 阅读器 (Advanced Regex PDF Reader)

这是一个基于 Python 开发的高性能桌面 PDF 阅读工具。它专为需要对 PDF 文档进行复杂信息提取和快速定位的用户设计，支持灵活的正则表达式搜索、预设管理以及流畅的连续页面滚动。

## ✨ 核心特性

- 🔍 **强大正则搜索**：支持标准正则表达式语法，全文深度搜索。
- 🎯 **精准定位跳转**：搜索结果列表展示，点击条目自动滚动至目标文字中心，并提供**深色高亮**区分。
- 📦 **正则预设管理**：支持手动添加、删除常用的正则模板，并为复杂正则命名（如“身份证号”、“日期格式”）。
- 📝 **模板即时微调**：选中预设后可在文本框内二次编辑，搜索以修改后的即时正则为准。
- 📜 **双显示模式**：
	- **单页模式**：传统的翻页阅读体验。
	- **连续模式**：类似网页的平滑滚动体验，支持跨页浏览。
- 🚀 **高性能渲染引擎**：采用\*\*虚拟滚动与懒加载（Lazy Loading）\*\*技术。无论 PDF 有多少页，仅渲染可见区域，内存占用极低。
- 🎡 **灵活缩放与导航**：支持 0.5x - 4.0x 无级缩放，支持页码直接跳转。

## 🛠️ 技术栈

- **语言**：Python 3.x
- **界面库**：PyQt5 (提供流畅的 GUI 交互)
- **内核库**：PyMuPDF (fitz) (高性能 PDF 解析与渲染)

## 📥 安装步骤

1. **克隆或下载本项目**到本地。
2. **安装依赖库**： 打开终端或命令行，执行以下命令：
	Bash
	```
	pip install PyQt5 PyMuPDF
	```
3. **运行程序**：
	Bash
	```
	python pdfsearch.py
	```

## 📖 使用说明

1. **打开文件**：点击左上角“打开 PDF 文件”选择你的文档。
2. **选择预设**：在下拉菜单中选择一个已有的正则名字。
3. **编辑并搜索**：
	- 选中的正则会出现在“当前正则”文本框中。
		- 你可以直接在该框内进行修改。
		- 点击“执行搜索”或按**回车键**开始搜索。
4. **浏览结果**：点击左侧列表生成的搜索结果，右侧 PDF 区域将自动跳转并高亮目标。
5. **管理预设**：使用“添加”按钮新建模板，使用“删除”按钮移除不再需要的条目。

## ⚙️ 配置文件说明

本程序已配置为**便携模式（Portable Mode）**。

所有的正则预设数据均保存在程序同级目录下的： `pdfsearch-config.ini`

你可以直接使用文本编辑器打开该文件进行手动备份或迁移。由于采用了 `.ini` 格式存储，即使更换电脑，只需带上该文件，你的所有正则预设都将得到保留。

## 🧠 技术原理：内存优化

为了防止大型 PDF 文件撑爆内存，程序实现了以下机制：

- **占位符布局**：连续模式下预先计算所有页面高度，但不渲染内容。
- **视野侦测**：仅当页面进入显示器的可视范围（含缓冲带）时，才触发像素渲染。
- **主动回收**：页面滑出视野后，自动调用图像清理函数释放内存，确保程序运行始终轻快。
